Exploring Cloud Technologies

Despre Curs

Obiective

Explorarea Tehnologiilor Cloud este un curs dedicat înțelegerii și aplicării tehnologiilor cloud moderne. Studenții vor învăța conceptele fundamentale ale cloud computing-ului, arhitecturile distribuite, serviciile cloud (IaaS, PaaS, SaaS), și vor dobândi experiență practică cu platforme cloud populare.

Conținut

  • Fundamentele cloud computing
  • Arhitecturi distribuite și microservicii
  • Servicii cloud: IaaS, PaaS, SaaS
  • Platforme cloud: AWS, Azure, Google Cloud
  • Containerizare și orchestrare (Docker, Kubernetes)
  • Securitate în cloud
  • Proiecte practice
